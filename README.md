## About

I like projects where gameplay, systems programming, and tools meet.

- Building NeoForge mods for Minecraft 1.21.1 with custom mechanics, world generation, content, data generation, and integrations.
- Experimenting with Rust-backed engines, native bridges, desktop tools, serial communication, and ESP32 firmware.
- Making focused web projects and study tools when the browser is the right interface.
- Keeping projects practical: build scripts, documentation, checks, and workflows matter as much as the idea.

## Selected Work

[**Nature**](https://github.com/iwosw/nature)  
NeoForge 1.21.1 mod with a Rust-backed procedural terrain engine, JNI bridge, native checks, water systems, biome climate sampling, wind and current queries, and cross-platform CI builds.

[**Vivarium Libera**](https://github.com/iwosw/Vivarium-Libera)  
Nature-focused Minecraft mod with plants, herbalist tools, custom blocks, plum wood, decorative jars, stream worldgen, localization, GeckoLib, Farmer's Delight integration, and data generation.

[**Cultio**](https://github.com/iwosw/cultio)  
Agriculture overhaul for Minecraft: seasonal growth, regional planting rules, crop diseases, crop rotation, weeds, ash-treated farmland, and an in-game field guide.

[**Animalis Agricultura**](https://github.com/iwosw/animalis-agricultura)  
Realistic livestock systems for Minecraft: sex, pregnancy, litters, nests, feeders, cow lactation, manure, inspection cards, breed variants, and server-side configuration.

[**Lumen et Signum**](https://github.com/iwosw/lumen-et-signum)  
Rust desktop control panel, serial monitor, EspScript runner, and companion firmware for ESP32 boards.

[**Unified-State-Exam-CS**](https://github.com/iwosw/Unified-State-Exam-CS)  
Static study site for Russian CS exam practice with tasks 1-27, short algorithms, code templates, search, SVG assets, and GitHub Pages deployment.
